# Cognifyz Full Stack Development Internship Project

## Project Overview
This project was developed as part of the **Cognifyz Technologies Full Stack Development Internship Program**.

It is a student registration web application built using **Node.js, Express.js, EJS, Bootstrap, JavaScript, HTML, and CSS**.

The project demonstrates frontend and backend full stack development concepts including form handling, validation, responsive UI design, dynamic DOM manipulation, and REST API development.

---

## Technologies Used
- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- REST API

---

## Features
- Student registration form
- Client-side form validation
- Server-side validation
- Temporary data storage
- Responsive Bootstrap UI
- Dynamic live preview
- Password strength validation
- REST API for student data
- CRUD operations
